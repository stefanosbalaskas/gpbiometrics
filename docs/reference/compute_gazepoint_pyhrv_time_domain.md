# Compute pyHRV-style time-domain summary

Compute pyHRV-style time-domain summary

## Usage

``` r
compute_gazepoint_pyhrv_time_domain(nni_ms, segment_seconds = 300)
```

## Arguments

- nni_ms:

  Numeric NN intervals in milliseconds.

- segment_seconds:

  Segment length for SDNN index and SDANN.

## Value

Data frame with time-domain HRV measures.
