# Create a time vector from NN intervals

Create a time vector from NN intervals

## Usage

``` r
create_gazepoint_pyhrv_time_vector(nni_ms, start_s = 0)
```

## Arguments

- nni_ms:

  Numeric NN intervals in milliseconds.

- start_s:

  Start time in seconds.

## Value

Numeric time vector in seconds.
