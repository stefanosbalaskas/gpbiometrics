# Compute phase-locking value between two signals

Compute phase-locking value between two signals

## Usage

``` r
compute_gazepoint_signal_phase_locking(x, y, sampling_rate_hz, band = NULL)
```

## Arguments

- x:

  First numeric signal.

- y:

  Second numeric signal.

- sampling_rate_hz:

  Sampling rate in Hz.

- band:

  Optional two-element frequency band.

## Value

Data frame with phase-locking value and circular phase difference.
