# Check whether NN intervals are inside a plausible interval

Check whether NN intervals are inside a plausible interval

## Usage

``` r
check_gazepoint_pyhrv_interval(nni_ms, min_ms = 250, max_ms = 2500)
```

## Arguments

- nni_ms:

  Numeric NN intervals in milliseconds.

- min_ms:

  Minimum plausible interval.

- max_ms:

  Maximum plausible interval.

## Value

Data frame with interval status.
