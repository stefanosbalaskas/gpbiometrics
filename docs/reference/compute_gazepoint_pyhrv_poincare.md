# Compute pyHRV-style Poincare measures

Compute pyHRV-style Poincare measures

## Usage

``` r
compute_gazepoint_pyhrv_poincare(nni_ms, plot = FALSE)
```

## Arguments

- nni_ms:

  Numeric NN intervals in milliseconds.

- plot:

  If TRUE, draw Poincare plot.

## Value

Data frame of Poincare measures.
