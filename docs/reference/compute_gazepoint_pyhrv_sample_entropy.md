# Compute sample entropy

Compute sample entropy

## Usage

``` r
compute_gazepoint_pyhrv_sample_entropy(nni_ms, m = 2L, r = NULL)
```

## Arguments

- nni_ms:

  Numeric NN intervals in milliseconds.

- m:

  Embedding dimension.

- r:

  Tolerance. If NULL, 0.2 \* SD is used.

## Value

Sample entropy.
