# Enhance pulse/PPG peaks using repeated local baseline removal

Enhance pulse/PPG peaks using repeated local baseline removal

## Usage

``` r
enhance_gazepoint_ppg_peaks(x, sampling_rate_hz, iterations = 2L)
```

## Arguments

- x:

  Numeric pulse/PPG signal.

- sampling_rate_hz:

  Sampling rate in Hz.

- iterations:

  Number of enhancement passes.

## Value

Enhanced numeric signal.
