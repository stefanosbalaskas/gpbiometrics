# Plot pyHRV-style heart-rate heatplot

Plot pyHRV-style heart-rate heatplot

## Usage

``` r
plot_gazepoint_pyhrv_hr_heatplot(nni_ms, time_bins = 20, hr_bins = 20)
```

## Arguments

- nni_ms:

  Numeric NN intervals in milliseconds.

- time_bins:

  Number of time bins.

- hr_bins:

  Number of heart-rate bins.

## Value

Invisibly returns heatplot data.
