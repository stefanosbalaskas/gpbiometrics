``` r
summarize_gazepoint_fixations_by_aoi(...)
```

- ...:

  Arguments passed to
  [`summarise_gazepoint_fixations_by_aoi()`](https://stefanosbalaskas.github.io/gpbiometrics/reference/summarise_gazepoint_fixations_by_aoi.md).
