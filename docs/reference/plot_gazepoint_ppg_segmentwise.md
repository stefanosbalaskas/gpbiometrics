# Plot segmentwise Gazepoint pulse/PPG measures

Plot segmentwise Gazepoint pulse/PPG measures

## Usage

``` r
plot_gazepoint_ppg_segmentwise(segmentwise, measure = "bpm")
```

## Arguments

- segmentwise:

  Object returned by process_gazepoint_ppg_segmentwise().

- measure:

  Measure column to plot.

## Value

Invisibly returns plotted data.
